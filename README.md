# README

This is my website [j2gl-bytes](https://j2gl.github.io/) Git repository.  You can check it here: https://j2gl.github.io/.

## Basic commands

To start the server:
```sh
hugo server
```

Then go to http://localhost:1313/

To create a new post: 
```sh
hugo new content/posts/{{filename}}.md
```
